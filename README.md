  ## DESCRIPTION
  ### This application uses PWA to generate an application that makes editing texts easier. With the use of PWA the user is able to install the application directly to their desktop to be used when internet access is not present. This text editor provides a useful tool when needing to make revisions to any type of texts.

  ## INSTALLATION
  ### To install this application the user must follow the heroku deployed link and press the install button on the navigation bar of the browser.

  ## USAGE
  ### This application has a easy to use interface. As the application opens it provides a workspace that includes a logo for JATE this is the space the user is able to delete and write their own necessary edits. The user can then install the application to their desktop for offline work using the button located on the search bar of the browser.
